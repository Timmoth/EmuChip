Chip-8 Emulator

CHIP-8 is a lightweight interpreted virtual machine originally designed for 8-bit microcomputers. It features a 4K memory space, 16 8-bit registers, a 64Ã—32 monochrome display, a stack for subroutines, and simple timers for delay and sound. This emulator replicates the CHIP-8 hardware in software, written in C# and compiled into WebAssembly for execution directly in the browser. Created by Tim Jones.
